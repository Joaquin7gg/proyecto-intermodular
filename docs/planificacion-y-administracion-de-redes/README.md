Este directorio recoge los documentos correspondientes al módulo de **Planificación y Administración de Redes**, dentro del proyecto intermodular de **1º ASIR**.

En este apartado se muestra la documentación de un repositorio relacionado con el diseño e implementación de una infraestructura de red segmentada mediante VLAN para un club con necesidades similares a las de una pequeña o mediana empresa.

## Descripción del proyecto

Este proyecto desarrolla la infraestructura de red de un club de fútbol ficticio, organizando la conectividad por departamentos y separando los servicios mediante VLAN.

La solución incluye una red distribuida en **dos plantas** y un **CPD central**, con un **switch multicapa** como núcleo de red, varios switches de acceso, servidores, red inalámbrica para empleados e invitados, impresoras compartidas y simulación de salida a internet.
