Este directorio recoge los documentos correspondientes al módulo de **Computación en la Nube**, dentro del proyecto intermodular de **1º ASIR**.

En este apartado se muestra la documentación de un repositorio relacionado con la implementación de una arquitectura híbrida, combinando infraestructura local (**on-premise**) con servicios en la nube mediante **Amazon Web Services (AWS)**.

El objetivo es complementar la infraestructura del club con una solución cloud que permita desplegar un servicio web público, almacenar archivos en la nube y alojar la base de datos del sistema, manteniendo al mismo tiempo los servicios internos críticos dentro del CPD local.


## Tecnologías y servicios utilizados

- **Amazon Web Services (AWS)**
- **Amazon EC2**
- **Amazon S3**
- **Amazon RDS (MySQL)**
- **Amazon VPC**
- **Amazon Linux**
- **Apache**
- **AWS CLI**
- **CloudShell**
