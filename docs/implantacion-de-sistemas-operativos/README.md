Este directorio recoge los documentos correspondientes al módulo de **Implementación de sistemas operativos**, dentro del proyecto intermodular de **1º ASIR**, donde se documenta el proceso de **implantación de una infraestructura cliente-servidor** para un club deportivo ficticio.  

El proyecto simula un entorno corporativo real en el que se despliegan sistemas operativos, se configura un dominio con Active Directory y se implementan distintos servicios de red.

Toda la infraestructura ha sido implementada en un **entorno virtualizado mediante VirtualBox**, permitiendo recrear un escenario completo de red empresarial con servidor central, equipos cliente y servicios corporativos. 

---

# Descripción del proyecto

El objetivo principal del proyecto es diseñar y documentar la implantación de un **entorno cliente-servidor completo**, incluyendo:

- instalación de sistemas operativos
- configuración de dominio mediante **Active Directory**
- gestión centralizada de usuarios y grupos
- control de permisos
- implementación de servicios de red
- validación del funcionamiento de la infraestructura

La infraestructura simula el entorno tecnológico de un club deportivo con diferentes departamentos y necesidades de acceso a recursos compartidos.

# Tecnologías utilizadas

Durante el desarrollo del proyecto se han utilizado las siguientes tecnologías:

**Sistemas operativos**

- Windows Server 2022
- Windows 11 Pro
- Ubuntu Desktop
- Kali Linux

**Servicios de red**

- Active Directory
- DNS
- Servidor de archivos
- Servidor de impresión
- Servidor FTP
- Intranet corporativa
- Escritorio remoto (RDP)

**Herramientas**

- VirtualBox
- PowerShell
- Internet Information Services (IIS)
